# daw-project

## Setup

Use Python 3.12 as the project interpreter.

Create and activate a virtual environment:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

Before running the project, install the required Python modules inside the activated virtual environment:

```powershell
pip install -r requirements.txt
```

## Run

```powershell
python server.py
```
